<P><B><A name=postavshiki>Расчеты с поставщикам</A></B></P><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; LINE-HEIGHT: 115%; mso-fareast-font-family: "Times New Roman"; mso-ansi-language: RU; mso-fareast-language: RU; mso-bidi-language: AR-SA'>Контроль расчетов с поставщиками выполняется следующим образом:</SPAN> 
<UL>
<LI><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; LINE-HEIGHT: 115%; mso-fareast-font-family: "Times New Roman"; mso-ansi-language: RU; mso-fareast-language: RU; mso-bidi-language: AR-SA'>на дату окончания анализируемого периода формируется таблица остатков по регистру «Расчеты с поставщиками»;</SPAN> 
<LI><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; LINE-HEIGHT: 115%; mso-fareast-font-family: "Times New Roman"; mso-ansi-language: RU; mso-fareast-language: RU; mso-bidi-language: AR-SA'>полученная таблица анализируется и некорректной считается ситуация, когда у контрагента, по одному и тому же договору и заказу имеются остатки долгов и авансов.</SPAN></LI></UL>
<P></P>Автоматическая корректировка выполняется по нажатию на гиперссылку «Выполнить зачет авансов». При этом происходит следующее: 
<UL>
<LI><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; LINE-HEIGHT: 115%; mso-fareast-font-family: "Times New Roman"; mso-ansi-language: RU; mso-fareast-language: RU; mso-bidi-language: AR-SA'>из регистра взаиморасчетов выбираются документы, изменившие состояние расчетов, за текущий период;</SPAN> 
<LI><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; LINE-HEIGHT: 115%; mso-fareast-font-family: "Times New Roman"; mso-ansi-language: RU; mso-fareast-language: RU; mso-bidi-language: AR-SA'>в каждом из отобранных документов производится программное заполнение расшифровки платежа для документов, формирующих авансы и предоплат для документов, формирующих долги.</SPAN></LI></UL>
<P><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; LINE-HEIGHT: 115%; mso-fareast-font-family: "Times New Roman"; mso-ansi-language: RU; mso-fareast-language: RU; mso-bidi-language: AR-SA'>Для документов вида «Ввод начальных остатков»&nbsp;автоматическая корректировка не предусмотрена, т.е. если в рамках одного и того же договора будет внесен остаток авансов и долгов, то данная ситуация будет считаться некорректной. При нажатии на гиперссылку «Выполнить зачет авансов» данная ситуация автоматически обрабатываться не будет.</SPAN></P>

<P><B><A name=nom_bez_spec>Номенклатура без спецификаций</A></B></P>
<P class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>Контролируется, чтобы для каждой номенклатурной позиции, для которой создана одна или более спецификаций, была указана спецификация по умолчанию. <o:p></o:p></SPAN></P>
<P class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>В случае обнаружения недочетов, пользователю предлагается:<o:p></o:p></SPAN></P>
<UL type=disc>
<LI class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; mso-list: l0 level1 lfo1; tab-stops: list 36.0pt"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>в случае если для конкретной номенклатурной позиции создана одна спецификация – <SPAN style="mso-spacerun: yes">&nbsp;</SPAN>установить её как спецификацию по умолчанию; <o:p></o:p></SPAN>
<LI class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; mso-list: l0 level1 lfo1; tab-stops: list 36.0pt"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>в случае если спецификаций больше чем одна – выбрать из списка необходимую.<o:p></o:p></SPAN></LI></UL>
<P><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; LINE-HEIGHT: 115%; mso-fareast-font-family: "Times New Roman"; mso-ansi-language: RU; mso-fareast-language: RU; mso-bidi-language: AR-SA'>Результаты исправлений могут влиять на результаты анализа подчиненных разделов. Влияние данного раздела на подчиненные будет описано отдельно для каждого из подчиненных разделов.</SPAN></P>

<P><B><A name=otchet_bez_spec>Отчеты переработчиков без спецификаций</A></B></P>

<P><B><A name=otchet_nesoot_spec>Отчеты переработчиков - несоответствие списаний спецификациям</A></B></P>

Контролироваться будут только документы, с указанной в них спецификацией продукции, поэтому выполнение корректировок по разделу «Отчеты переработчиков без спецификаций» важно для наиболее качественного анализа по данному разделу. <o:p></o:p></SPAN></P>

<P><B><A name=zakup_prices>Анализ закупочных цен</A></B></P>

<P><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; LINE-HEIGHT: 115%; mso-fareast-font-family: "Times New Roman"; mso-ansi-language: RU; mso-fareast-language: RU; mso-bidi-language: AR-SA'>В данном разделе не предполагается каких-либо действий по автоматическому исправлению, но для каждой позиции доступен отчет, показывающий динамику изменения цен – по нажатию на гиперссылку «Результат анализа».</SPAN>

<P class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>Например: <o:p></o:p></SPAN></P>
<UL type=disc>
<LI class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; mso-list: l0 level1 lfo1; tab-stops: list 36.0pt"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>если расходная накладная была введена на основании заказа покупателя, то заказ будет указан в поле «Основание» данной расходной, и при этом заказ будет считаться подчиненным расходной накладной; <o:p></o:p></SPAN>
<LI class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; mso-list: l0 level1 lfo1; tab-stops: list 36.0pt"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>при вводе приходной накладной в табличном поле «Запасы», содержащем 2 строки, были указаны 2 заказа поставщику – в данном случае оба заказа будут рассматриваться как подчиненные приходной накладной. <o:p></o:p></SPAN></LI></UL>

<P><B><A name=dds>Статьи движения денежных средств</A></B></P>
<P class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>В ходе анализа будет выполнена проверка соответствия движений денежных средств (платежей и поступлений ДС) предопределенным группам («Платежи», «Поступления») статей движения денежных средств за анализируемый период. <o:p></o:p></SPAN></P>
<P class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>Алгоритм проверки следующий: <o:p></o:p></SPAN></P>
<UL type=disc>
<LI class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; mso-list: l0 level1 lfo1; tab-stops: list 36.0pt"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>если некоторая статья ДДС входит в предопределенную группу «Платежи», то предполагается, что по данной статье могут быть только расходные движения ДС. Исключением считается ситуация возврата денег поставщиком (статья ДДС «Оплата поставщикам» и соответствующие виды операций в документах «Поступление в кассу» и/или «Поступление на счет») – в случае невыполнения описанных условий пользователю будет выведено сообщение о возможной ошибке учета; <o:p></o:p></SPAN>
<LI class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; mso-list: l0 level1 lfo1; tab-stops: list 36.0pt"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>если некоторая статья ДДС входит в предопределенную группу «Поступления», то предполагается, что по данной статье могут быть только приходные движения ДС. Исключением считается ситуация возврата денег покупателю (статья ДДС «Оплата от покупателей» и соответствующие виды операций в документах «Расход из кассы» и/или «Расход со счета») – в случае невыполнения описанных условий пользователю будет выведено сообщение о возможной ошибке учета; <o:p></o:p></SPAN>
<LI class=MsoNormal style="MARGIN: 0cm 0cm 10pt; LINE-HEIGHT: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; mso-list: l0 level1 lfo1; tab-stops: list 36.0pt"><SPAN style='FONT-SIZE: 12pt; FONT-FAMILY: "Times New Roman","serif"; mso-fareast-font-family: "Times New Roman"; mso-fareast-language: RU'>если некоторая статья ДДС не входит ни в одну из вышеперечисленных предопределенных групп статей ДДС, то предполагается, что по данной статье, сумма приходных движений должна быть равна сумме расходных движений – в случае невыполнения данного условия пользователю будет выведено сообщение о возможной ошибке учета. <o:p></o:p></SPAN></LI></UL>
